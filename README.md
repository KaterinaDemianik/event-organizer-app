### Курсова робота

- Тема: Створення веб-додатку для організації подій із застосуванням патернів проєктування. 
- Мета: розробити веб-додаток для створення, перегляду та управління подіями.  
У межах роботи буде реалізовано основні функції CRUD (створення, читання, оновлення, видалення) із застосуванням патернів проєктування для покращення структури та гнучкості коду
- Ідея: проєкт продемонструє, як архітектурні підходи та шаблони проєктування допомагають створювати зручні, масштабовані та зрозумілі веб-застосунки
